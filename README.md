FLUX PoS

FLUX PoS is a modern point-of-sale and inventory management system built with the Laravel framework. It is designed for small retail stores, school supply shops, and other businesses that need an efficient way to manage products, sales, customers, and reports.

Features
Product and inventory management
Sales processing with printable receipts
Customer management
Real-time dashboard analytics
Low stock alerts
Reports and insights
User authentication
Tech Stack
Laravel
MySQL
Blade
Tailwind CSS
Chart.js
Installation
git clone <repository-url>
cd flux-pos
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
